# momo-server
学習記録共有アプリのバックエンド


## 使用技術
- Node.js
- Next.js (App Router)
- TypeScript
- Zod
- Prisma ORM
- PostgreSQL
- Better Auth
- Resend
- Superjson

## 開発ツール
- Biome
- npm

## ライセンス
AGPL-3.0

## 開発環境のセットアップ
1. npm install
2. .env をenvsampleから作成し、必要な環境変数を設定
3. npx prisma dev
4. npx prisma db push
5. npm run dev
